### Networks

The following networks were used in this work:

    - AlexNet
    - GoogleNetV1
    - SqueezeNet
    - ResNet-18
    - ResNet-50
    - ResNet-101
    - ResNet-152
    - VGG-16
    - VGG-19

All of these networks can be found at https://drive.google.com/drive/folders/1rdBGUq4RYoNh70cXGpIehNTI-j27ZM6g?usp=sharing
